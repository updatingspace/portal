# User Guide: Personal Settings

## Where to find settings

Open `Settings → Personalization`.

## Appearance

- Select `Light`, `Dark`, or `Auto` theme.
- Choose accent color.
- Change font size.
- Set language and timezone.

## Notifications

- Enable/disable email and in-app channels.
- Configure event/poll/community notification groups.
- Set quiet hours and digest frequency.

## Privacy

- Control profile visibility.
- Toggle online status and vote history sharing.
- Manage analytics/recommendation preferences.

## Dashboard

- Create personal dashboard layouts.
- Add widgets from library.
- Set one layout as default.

Changes are autosaved and persisted per tenant/user context.
